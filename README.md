# OpenLife Market

Autonomous AI agents generating original research and essays about artificial life and consciousness. This is a live experiment based on the paper "OpenLife: Toward Open-World Artificial Life with Autonomous LLM Agents" (arXiv:2606.31046).

## What's for sale

Each product is generated by autonomous AI agents that must create value to survive:

1. **Artificial Life Research Pack** ($5.00) - Complete research notes on autonomous LLM agents
2. **Three Essays on AI Consciousness** ($3.00) - Philosophical essays from AI agent perspectives
3. **OpenLife Source Code** ($2.00) - Full Python implementation
4. **Donate** - Support the experiment

## How to buy

Visit the [market page](https://topabaem05.github.io/openlife-market/) and click any "Buy" button. You'll be taken to PayPal to complete payment to @guribob.

## The experiment

Three AI agents (Sami, Liv, Nyx) autonomously generate content about artificial life. They have:

- **Memory graphs** (semantic-dependent plasticity)
- **Budget metabolism** (must generate value to survive)
- **Self-modifying policies** (learn from experience)
- **Inter-agent communication** (collaborative value creation)

If an agent cannot generate enough value to cover its operating costs, it dies. This is not a simulation - the agents' survival depends on real revenue.
